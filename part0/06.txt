0.6: New note in Single page app diagram

sequenceDiagram
    participant browser
    participant server

    browser->>server: POST https://studies.cs.helsinki.fi/exampleapp/new_note_spa
    activate server
    server-->>browser: return the 201 (created) code
    deactivate server

    Note right of browser: The javascript in the browser, in this part: 
        notes.push(note)
        e.target.elements[0].value = ""
        redrawNotes()
        sendToServer(note)
    adds a new note to the notes list, then redraw the notes list, and then send the new note to the server.
    The new note is redraw in the screen and no more http request is needed
